JSON-RPC Connection

Web3 clients query the network state via JSON-RPC payloads sent to blockchain nodes. 

// Get ETH balance of account
curl https://mainnet.infura.io/v3/PROJECT_ID \
-X POST \
-H "Content-Type: application/json" \
--data '{
  "jsonrpc": "2.0",
  "method": "eth_getBalance",
  "params": ["0x742d35Cc6634C0532925a3b844Bc454e4438f44e", "latest"],
  "id": 1
}'

Solidity Smart Contract

Ethereum supports complex state transitions via code run in the EVM. 

// SPDX-License-Identifier: MIT
pragma solidity ^0.8.20;
contract TokenStore {
    mapping(address => uint256) public balances;
    function deposit() public payable {
        balances[msg.sender] += msg.value;
    }
}

Public vs Private Keys Cryptography

Asymmetric encryption secures wallet authentication and message signing.

Aspect Public Key (Address) Private Key (Secret)
Analogy Your Email Address / Mailbox Your Mailbox Key / Password
Visibility Shared publicly on-chain Strictly secret (Never share)
Role Receiving tokens / tracking assets Authorizing & signing transfers

ECDSA Signing Math

Web3 uses Elliptic Curve Digital Signature Algorithm (secp256k1) to authorize operations. 

// Private Key (d): Random 256-bit integer
// Public Key (Q): Q = d * G (point on curve)
// Signature (r, s) generated from message (z):
// k = random_nonce()
// R = k * G (point on curve)
// r = R.x coordinate (mod n)
// s = k⁻¹ * (z + r * d) (mod n)
